Quantile-Quantile Plots

Mon, Mar 1, 2021 Quantile-quantile plots are a useful tool for determining whether a measure is normally distributed. From Wikipedia: Q–Q (quantile-quantile) plot is a probability plot, which is a graphical method for comparing two probability distributions by plotting their quantiles against each other. First, the set of intervals for the quantiles is chosen. A point (x, y) on the plot corresponds to one of the quantiles of the second distribution (y-coordinate) plotted against the same quantile of the first distribution (x-coordinate).

Random Numbers and Seeds in R

Thu, Feb 18, 2021 To explore how R generates random numbers, we will use the rnorm function. This function draws a random number from a normal distribution with a mean = 0 and standard deviation = 1 (though these can be changed with the mean and sd parameters). With n = 1 we will get one random number. rnorm(n = 1) ## [1] 0.6308552 rnorm(n = 1) ## [1] 1.36274 Each time you run the command you will get a different number.
